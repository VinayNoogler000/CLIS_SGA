# Command Line Interfaces and Scripting - SGA

This repository contains my Staff Graded Assignment (SGA) for the **Command Line Interfaces and Scripting** course, completed as part of the BSc in Computer Science program at BITS Pilani. 

**Note:** This is an academic coursework submission rather than a standalone development project. It demonstrates practical competency in Linux environments, command-line operations, system navigation, and shell scripting.

## Repository Structure

The assignment is organized into five distinct directories:

*   `Question1/` to `Question5/`: Each folder contains the complete solution for a specific assignment question.
*   `solution.md`: Located within each question folder, consists of the exact commands executed, the expected terminal outputs, conceptual explanations for every action, and execution screenshots.
